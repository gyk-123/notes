## langchain

langchian 核心价值就是标准化，让 AI 变得更加简单。

数据向量化、问题向量化、相似性搜索（基于向量）、接入 llm（openapi）润色，生成特定的结果。

llm 做整理、分析，这也是大语言模型应该做，并且擅长的的事情。
